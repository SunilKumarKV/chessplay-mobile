# ChessPlay Store Listing Draft

## Short Description

Play chess online, train with puzzles, and track your progress with ChessPlay.

## Full Description

ChessPlay is a native mobile chess app for playing real games, training tactics, and staying connected with other players.

Play online multiplayer games, create private rooms, join by room code, review your move history, climb the leaderboard, and practice tactical puzzles. ChessPlay is built for mobile with touch-based board controls, secure account sessions, dark and light themes, and real-time Socket.IO gameplay.

Some advanced features depend on backend rollout, including push notification delivery, reset-password deep link completion, email verification deep link completion, and fully automated store-compliant mobile purchases.

## Feature List
