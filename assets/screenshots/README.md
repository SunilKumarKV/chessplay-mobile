## Google Play Required Sets

Capture at least 4 screenshots for phone:

- 1080 x 1920 px portrait, or larger 9:16 portrait
- No transparent status/navigation artifacts
- Show real app screens, not marketing-only graphics

Recommended ChessPlay set:

1. Home dashboard
2. Online play / room creation
3. Live chess board with promotion or chat visible
4. Profile / leaderboard
5. Puzzles / training

## App Store Recommended Sets

Capture screenshots for:

- 6.7-inch iPhone
- 6.5-inch iPhone if required by App Store Connect
- 5.5-inch iPhone if required by App Store Connect
- iPad only if tablet listing is enabled
