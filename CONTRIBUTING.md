# Contributing to ChessPlay Mobile

Thank you for contributing.

## Local Setup

```bash
npm install
cp .env.example .env
npm run typecheck
npm run lint
```

## Contribution Rules

- Do not commit real API endpoints, secrets, or credentials.
- Keep auth/session handling secure.
- Do not weaken SecureStore usage.
- Test mobile UX on relevant devices or emulators.
- Keep production release docs accurate.
- Respect store submission/privacy requirements.

## Pull Requests

Use the pull request template and include screenshots or recordings for UI changes.
